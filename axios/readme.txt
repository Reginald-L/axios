https://github.com/axios/axios

Installing
    Using npm:
        $ npm install axios
    Using bower:
        $ bower install axios
    Using yarn:
        $ yarn add axios
    
    在项目中通常使用的是npm与yarn

    Using jsDelivr CDN:
        <script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
    Using unpkg CDN:
        <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
    在练习中可以使用CDN
